# Tier 1 Edge Function Deletion – 7 Feb 2026

## Context

The Notewell project accumulated **232 Supabase Edge Functions**. During publishing, Lovable deploys all functions via the Supabase Management API, which enforces a rate limit of approximately **30 requests per minute**. This caused **HTTP 429 (Too Many Requests)** errors, resulting in partial or failed deployments.

A full audit was conducted to identify functions with **zero callers** in `src/`, other edge functions, or any runtime code. These were classified as **Tier 1 – High Confidence Dead**.
